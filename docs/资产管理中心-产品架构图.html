车辆数字档案中心

建立一车一档，沉淀车辆、合规、电池、维保和事故数据。

  • 基础属性 VIN / 三电 / 上牌
  • 合规证照 证件归档 / 到期提醒
  • 电池健康 SOH / 异常画像
  • 维保事故 工单 / 出险 / 理赔
经营核心

租赁账务与资产风控中心

把租赁合同转成自动计费、对账、催收和资产风险处置规则。

  • 合同台账 起租 / 到期 / 免租
  • 计费规则 月租 / 里程 / 车电分离
  • 账单对账 自动算费 / 三方对账
  • 逾期风控 提醒 / 预警 / 远程限制
资产退出

残值评估与退役处置中心

评估车辆残值，管理二手处置、报废审批和动力电池退役流向。

  • 残值模型 里程 / SOH / 事故 / 行情
  • 处置资产池 年限 / SOH 阈值
  • 竞价审批 拍卖 / 报废 / 多级审批
  • 电池追溯 梯次利用 / 回收报告

资产管理中心

1 资产规模 在租、闲置、待处置
2 健康状态 SOH、维保、事故
3 回款风险 应收、逾期、账龄
4 残值退出 估值、拍卖、退役
